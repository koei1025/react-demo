# react-php-isomorphic-example

部屬
---
**安裝相依套件 & 打包 components**
```
npm install
npm run watch
```
**運行 http server (via PHP)**
```
php -S 0.0.0.0:8080 -t .
```
`http://0.0.0.0:8080/`
 
環境
---
* PHP 5.5.9 
* V8Js 0.1.3 (PHP extension)
